# Changelog

All notable changes to TMUX Manager will be documented in this file.

## [0.1.0] - 2026-04-24

- Initial independent TMUX Manager release prepared from an MIT-licensed upstream extension.
- Reset version to `0.1.0` for the new publisher and marketplace identity.
- Renamed extension metadata, commands, views, and storage keys to TMUX Manager/tmuxManager.
- Uses clean-slate `tmuxManager.*` storage keys; saved connection data from upstream builds is intentionally not migrated.
- Replaced extension icons with original TMUX Manager artwork.
